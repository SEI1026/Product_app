デバイス機能の設定
お使いのデバイスによって、機能バージョン/デバイス機能を設定する必要があります。
機能バージョン/デバイス機能によって、使用できる機能が異なります。デバイスがサポートしている機能をお使いになるために、機能バージョン/デバイス機能を設定してください。
1
デバイスのパネルからステータスプリントを出力 → 「コントローラ機能ID」と「機能バージョン」を確認
詳細は、デバイス付属のマニュアルを参照してください。
ステータスプリントの出力方法は、お使いのデバイスの機種によって異なります。
デバイス機能の設定のみ行う場合には、ステータスプリントの出力は不要です。
2
［］（スタート）メニューから［デバイスとプリンター］を選択
Windows 8/8.1/Server 2012/Server 2012 R2：デスクトップで画面右側のチャームを表示したあと、［設定］→［コントロールパネル］→［デバイスとプリンター］をクリックまたはタップ
Windows Vista：［］（スタート）メニューから［コントロールパネル］－［プリンタ］を選択
Windows XP Professional/Server 2003：［スタート］－［プリンタとFAX］を選択
Windows XP Home Edition：［スタート］－［コントロールパネル］－［プリンタとその他のハードウェア］－［プリンタとFAX］を選択
3
お使いのデバイスのアイコンを右クリックして、［プリンターのプロパティ］（または［プロパティ］）をクリックします。
4
[デバイスの設定]タブ → [デバイス機能]をクリック
5
手順1で確認した機能バージョン、またはデバイス機能を設定 → [OK]をクリック
機能バージョンを変更した場合は、お使いのデバイスの機能バージョンの設定によって、いくつかの機能が使えなくなる場合があります。
6
［OK］をクリック
設定した機能が有効になります。
