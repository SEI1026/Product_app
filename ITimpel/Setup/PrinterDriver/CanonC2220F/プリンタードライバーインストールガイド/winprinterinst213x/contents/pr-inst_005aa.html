オプション設定
デバイスに装着されているオプション（給紙カセット、フィニッシャーなど）の機能を使用して印刷するには、ドライバーでのオプション設定が必要です。以下の手順で、オプション構成を確認/設定してください。
重要
お使いのデバイスによっては、以下のいずれかが必要です。
Novell NetWare IPX/SPX Client Software
TCP/IPプロトコル
コンピューターにAdministratorsのメンバーとしてログオンしてください。
メモ
通常はドライバーインストール時に自動的に設定されます。あとからオプションを追加したときなど、自動的に設定が反映されない場合に、本操作を行ってください。
1
［］（スタート）メニューから［デバイスとプリンター］を選択
Windows 8/8.1/Server 2012/Server 2012 R2：デスクトップで画面右側のチャームを表示したあと、［設定］→［コントロールパネル］→［デバイスとプリンター］をクリックまたはタップ
Windows Vista：［］（スタート）－［コントロールパネル］－［プリンタ］を選択
Windows XP Professional/Server 2003：［スタート］－［プリンタとFAX］を選択
Windows XP Home Edition：［スタート］－［コントロールパネル］－［プリンタとその他のハードウェア］－［プリンタとFAX］を選択
2
設定を行うデバイスのアイコンを右クリック → [プリンターのプロパティ](または［プロパティ］)をクリック
3
［デバイスの設定］タブをクリック → デバイス情報の取得状況を確認
［自動］：インストール時にデバイス情報を取得して、オプション構成が自動設定されています。設定内容を確認したあと、［OK］をクリックして終了します。
［手動］：デバイス情報は取得されていません。手順4に進んで、オプション設定を行います。
4
［デバイス情報取得］をクリック
5
「デバイス情報を取得しますか？」というメッセージが表示されたら、［はい］をクリック
情報取得できた場合：オプション構成が自動設定されました。設定内容を確認したあと、［OK］をクリックします。
情報取得できなかった場合：手動でオプション設定を行います。デバイスに装着されているオプションを選択したあと、［OK］をクリックします。
メモ
共有プリンター環境で、デバイスの環境設定を自動で行う場合は、プリントサーバーにCanon Driver Information Assist Serviceをインストールする必要があります。詳細は、「Canon Driver Information Assist Serviceのインストール」を参照してください。
ローカル接続（USBポートなど）でデバイスをお使いの場合は、デバイスの環境設定を自動で行うことはできません。
